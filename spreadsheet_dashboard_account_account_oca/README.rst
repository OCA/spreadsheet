This module adds some dashboards for accounting and finance BI.

They have been rescued from some original Odoo SA code that were moved to
enterprise later, and then they are evolving isolatedly.

**Table of contents**

.. contents::
   :local:

Usage
=====

#. Go to *Dashboards* application.
#. There, you will find at the left two new options inside "Finance" section:

   * Accounting
   * Benchmark
#. Click on them to see the KPIs and BI information about accounting and
   finance.
